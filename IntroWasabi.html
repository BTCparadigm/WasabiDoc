# WasabiDoc

Wasabi Wallet is an open-source, non-custodial, privacy focused Bitcoin wallet for desktop. It implements trustless coin shuffling: Schnorrian CoinJoin.

This is Wasabi Wallet's open source documentation repository. Here you will find information about the nuances of privacy in Bitcoin, how Wasabi solves some of the existing problems, and how you can use this tool to defend yourself. If you have a question regarding the documentation, please open an issue. If you have an answer to a question, please open a pull request.
