# Build from source code

# Get The Requirements

  1. Install Git
  2. Install .NET Core 2.2 SDK for "Building Apps" (Note, you can disable .NET's telemetry by typing on Linux and OSX: export DOTNET_CLI_TELEMETRY_OPTOUT=1 or on Windows set DOTNET_CLI_TELEMETRY_OPTOUT=1.)

# Get Wasabi

Clone & Restore & Build

git clone https://github.com/zkSNACKs/WalletWasabi.git
cd WalletWasabi/WalletWasabi.Gui
dotnet build

# Run Wasabi

Run Wasabi with dotnet run from the WalletWasabi.Gui folder.

# Update Wasabi

git pull
