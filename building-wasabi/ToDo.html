# ToDo's in Wasabi

Wasabi Wallet is libre and open source software. This means that contributions are not just very welcome, but absolutely needed! If you are considering to support the project, please checkout the contribution checklist for how to best get started!

This document is for providing a one place stop for anyone interested in the short term ToDo's and long term goals. When you are ever bored and want to be productive, why not pick up one of these issues?

# Short Term ToDo's

  • Manually change the anonset shield
  • RAM usage improvements
  • Pay to many
  • Multi wallet support
  • Improve cluster history
  • Address typo location detector
  • Sweap private key
