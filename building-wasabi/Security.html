# Security Policy

Feel free to report a vulnerability as a GitHub issue only if the vulnerability you found does not compromise users' privacy or security.

If it does, then pay great care to responsible disclosure. Send an email to adam.ficsor73@gmail.com, preferably using PGP encryption: 21D7 CA45 565D BCCE BE45 115D B4B7 2266 C47E 075E
